1- Rodar main

2- Digitar "localhost" no local correspondente a IP

3- Digitar a porta "502"

4- Leitura:
4.1- Escolha qual tipo de leitura quer realizar
4.2- Clique em ler
4.3 - O valor vai aparecer na parte de valor leitura
4.4- Para leitura contínua marque o checkbox

5- Escrita
5.1 - Clique no botão referente ao que quer escrever
5.2 - Escreva o valor no local referente a escrita
5.3 - Para caso de escolher Bit, escreva também a posição do bit que será modificado, no ícone referente a posição
